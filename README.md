# scholar-paper-mcp

Hybrid MCP server for Semantic Scholar. Persistent SQLite cache, offline fallback, vector search. Pairs with document-writing skill for hackathon, proposal, thesis, and article workflows.

Status: planning phase. See `docs/PLAN.md`.

Stack: Python 3.13, FastMCP, httpx, SQLite + sqlite-vec, ONNX MiniLM, pytest + ruff + ty.

Upstream reference: https://github.com/akapet00/semantic-scholar-mcp
